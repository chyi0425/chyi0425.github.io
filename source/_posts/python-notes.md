## 基础

### list

有序集合，可变

```Python
>>> classmates = ['Michael', 'Bob', 'Tracy']
>>> classmates.append('Adam')
>>> classmates.insert(1, 'Jack')
>>> classmates.pop()
>>> classmates.pop(1)
>>> classmates[1] = 'Sarah'
```

### tuple

```Python
>>> classmates = ('Michael', 'Bob', 'Tracy')
```

有序集合，不可变
